# Focusboard

A focused Kanban-style board with WIP limits, per-card emoji icons, and customizable column colors/backgrounds.

## Quick start

```bash
npm install
npm run dev
```

Open `http://localhost:5173/`.

## Scripts

- `npm run dev` - Start the Vite dev server.
- `npm run build` - Typecheck and build for production.
- `npm run typecheck` - Run TypeScript typechecking.
- `npm run lint` - Run ESLint.
- `npm run preview` - Preview the production build.

## Customization

- Background image, column colors, and column icons are managed in the in-app Settings panel.
- Card emoji icons can be set per card in the card editor.
